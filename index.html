What this project is / 概要

English This project is a linguistic entry gate for Prema OS. It reorganizes a limited set of Sanskrit-based words as a practical vocabulary for education, self-understanding, and AI dialogue.

日本語 Prema OS は、倫理的な言語のための実験的フレームワークです。サンスクリット語に基づく限られた数の言葉を、教育・内省・AIとの対話のための実用的な語彙として再編することを目指しています。その目的は伝統的な学問の置き換えではなく、自己理解と今後のAI設計のための道具として再配置する試みです。

Why Sanskrit in Japan? / なぜ日本でサンスクリットか

English Sanskrit is not foreign to Japan. Through Kūkai, Shingon, and Buddhist vocabulary, it has been rooted in Japanese culture for over a millennium. We build on this unique history of reception to explore how this linguistic layer can serve modern education.

日本語 それは、サンスクリット語が日本にとって決して異質なものではないからです。空海、真言、悉曇文字を通じて、すでに千年以上にわたって日本文化の中に静かに根を下ろしてきました。本プロジェクトは、日本におけるサンスクリット受容の歴史の上に立ちながら、その言語層が現代の市民的想像力のためにもう一度役立ちうるかを探ります。

The Foundation / このプロジェクトの核心

English This project is grounded in a single question: can language truly preserve meaning in the age of AI? We call this inquiry "Linguistic Physics." Sanskrit's root system (Dhātu) — where meaning is preserved down to the smallest unit — is not a relic of the past, but a 2,500-year-old answer to a problem we are only now beginning to ask. The Cube App, the Mandala, and every tool here are built on this foundation. This document is not a completed theory — it is a declaration that sets the question. If the question is right, the answers will follow.

日本語 このプロジェクトは一つの問いを土台にしています。AI時代において、言語は本当に意味を保存できるのか。私たちはその問いを「言語物理学」と呼びます。サンスクリット語根（Dhātu）は過去の遺物ではなく、私たちがようやく問い始めた問題への、2500年前の解答です。キューブ・アプリも、マンダラも、このサイトのすべての道具は、この問いを基盤としています。これは完成された論文ではなく、問いを立てるための宣言書です。問いの設定が正しければ、答えは後から来る。
